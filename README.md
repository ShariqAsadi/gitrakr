## Built With
  - [React](https://reactjs.org/)
  - [Styled Components](https://styled-components.com/)
  - [Chart.js](https://www.chartjs.org/)
  - [GitHub Rest API](https://docs.github.com/en/free-pro-team@latest/rest)

## Installation

1. Install dependencies

```bash
npm install
```

2. Run the server

```bash
 npm start
```
